# 更新日志

本文档记录有声读物智能生成系统的重要功能变化。

日志格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号根据现有 Git 历史整理；历史版本尚未创建对应的 Git Tag。

## [未发布]


### 新增

- 创作工作台新增语言选择器，支持将中文文本翻译为 6 种语言后合成语音（en/ja/ko/fr/de/zh）

### 修复

- 修复 ackend/app/api/admin.py 和 eports.py UTF-16 编码问题，转换为 UTF-8 避免 null bytes 语法错误
- 修复 .env 中 MySQL 连接密码为空导致的启动失败

### 新增

- 使用 `jieba` 进行中文分词，使超长文本优先在完整词语边界处分段。
- 增加中文词组完整性、标点归属和长文本分段测试。

### 优化

- 保留段落、句号、问号、分号和逗号对应的不同停顿时长。
- 词语边界产生的技术性分段使用较短停顿，减少不自然的断句和连读。
- 长文本支持分段合成、失败重试、并发控制和音频拼接。

## [0.3.0] - 2026-06-18

### 新增

- 新增独立登录页面、个人资料页面、手机号绑定和头像更新功能。
- 新增 SQLAlchemy 音色数据库模型、声音 CRUD 接口和 MySQL 初始化脚本。
- 新增 Edge-TTS 音色试听接口以及试听音频缓存。
- 新增 FastAPI 接口测试和数据库结构测试。

### 变更

- 页面图标从 Emoji 全面替换为 Lucide 矢量图标。
- 合并前端登录系统、数据库音色服务和原有全文 TTS 后端。
- 前端开发服务器增加 `/api`、`/static`、`/audio` 和 `/media` 代理。

## [0.2.0] - 2026-06-16

### 新增

- 新增 FastAPI 后端和 Edge-TTS 全文语音合成功能。
- 新增场景模板、中文音色库、文本预处理和敏感词检测接口。
- 新增作品列表、作品详情、作品删除和 MP3 文件访问接口。
- 新增 JSON 作品元数据存储和本地音频文件存储。
- 前端工作台、音色库和“我的作品”页面接入后端 API。
- 音频播放器支持真实播放、暂停、进度、音量和下载。

## [0.1.0] - 2026-06-16

### 新增

- 初始化 Vue 3、Vite、Vue Router 和 Pinia 前端项目。
- 完成首页、创作工作台、音色库和“我的作品”基础页面。
- 完成场景选择、文本输入、音色选择和参数设置交互原型。

## [0.0.1] - 2026-06-12

### 新增

- 创建项目仓库和初始目录结构。
